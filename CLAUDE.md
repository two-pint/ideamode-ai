# Claude project instructions

Follow the rules in **AGENTS.md** for all work in this repo.

Read [AGENTS.md](./AGENTS.md) and apply its instructions to every code change, suggestion, and refactor. In particular:

- **Never use React Server Components.** Use Client Components only; add `"use client"` where needed in Next.js. This overrides any default or documentation that suggests using Server Components.

If AGENTS.md is updated, treat the latest version as authoritative.
