# @ideamode/assets

Shared brand assets for IdeaMode. Consumed by `apps/web` (Next.js) and `apps/mobile` (Expo) via workspace dependency.

## Files

| File | Role | Web | Mobile |
|------|------|-----|--------|
| `ideamode_icon.svg` | App icon (symbol only) | Favicon, PWA icons, compact nav | App icon, tab bar |
| `ideamode_logo.svg` | Full logo (wordmark + symbol) | Header, login/marketing, footer | Splash, auth screen, header |

See [Shared assets plan](../../docs/shared-assets-plan.md) for full usage and how to add more assets.
